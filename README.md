# ELT Pipeline for Financial News & BERT

## Contents

- **Submission.ipynb** — Main Jupyter notebook (pipeline and analysis).
- **Submission.html** — Exported HTML of the notebook.
- **flask.py** — Flask script for the web interface.
- **parquet_snowflake.ipynb** — Parquet and Snowflake-related work.

## Setup & run

- Python 3.x.
- Install dependencies as needed by the notebooks or a `requirements.txt` if present.
- To run the Flask app: `python flask.py`

## Data

- Large data (e.g. `layer/`) and model assets (e.g. `models/`) are not in this repo; set them up locally or in your environment as needed.
